# Custom Search v2.0.0

[Перейти на сайт](https://mihail174.ru/docs/developments/custom_search)

## Основные возможности
1. __Изменение плейсхолдера__

   Можно задать свой плейсхолдек для разных устройств (десктоп/телефон). Смена происходит при разрешении 768px.


2. __Кнопка "очистить поле"__

   Очищает введенное содержимое в текстовом поле


3. __Менять брейкпоинт можно через отдельный пункт в настройках.__
 

4. __Выбор вида поиска в шапке сайта.__

      Краткий или полный вид отображения и скольжение при выборе краткого вида.

## Использование

      new CustomSearch();

## Опции

| Опция | Тип | По умолчанию | Детальное описание                                                                                                                                                                                                                                                                                                            |
|-------|-----|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `clear` | boolean | `true` | Кнопка очистить введеное значение в поле                                                                                                                                                                                                                                                                                      |
| `placeholderDesktop` | string | `'Введите наименование'` | Текст плейхолдера для десктопной версии                                                                                                                                                                                                                                                                                       |
| `placeholderMobile` | string | `'Поиск'` | Текст плейхолдера для мобильной версии                                                                                                                                                                                                                                                                                        |
| `placeholderBreakpoint` | number \| string \| array | `767` | Точка при которой будет происходить изменение плейсхолдера. <p>Number</p><pre>placeholderBreakpoint: 767</pre><p>String</p><pre>placeholderBreakpoint: "mobile"</pre><p>Array</p><pre>placeholderBreakpoint: ["mobile", "laptop"]</pre><p>Названия указываются при наличии скрипта 'Типы устройств + Диапазоны устройств'</p> |
| `headerType` | `'short'` \| `'full'` | `'full'` | Указывается тип поиска                                                                                                                                                                                                                                                                                                        |
| `headerSlideDirection` | `'left'` \| `'right'` | `null` | В какую сторону раскрывать поиск <p>При наличии параметра <code>headerType: 'short'</code></p>                                                                                                                                                                                                                                |
| `headerBreakpoint` | array |  | Диапазоны разрешения. <pre>{<br>    breakpoint: [767, 1023] \| ['mobile', 'tablet'],<br>    type: 'short' \| 'full',<br>    slideDirection: 'left' \| 'right'<br>}</pre><p><code>headerType</code> и <code>headerSlideDirection</code> игнорируются</p>                                                                       |
