# Custom Search v3.0.0

[Перейти на сайт](https://mihail174.ru/docs/developments/custom_search)

## Основные возможности
1. __Изменение плейсхолдера__

   Можно указывать для определенных разрешений, при использовании скрипта [CustomDevices](https://github.com/mihail-174/custom_devices).


2. __Кнопка "очистить поле"__

   Очищает введенное содержимое в текстовом поле

## Использование

### По умолчанию:

      new CustomSearch();


### Свои настройки:

Влючение или отключение кнопки "очистить":

      new CustomSearch({
         clear: false
      });

Плейсхолдер для всех разрешений сразу:

      new CustomSearch({
         placeholder: 'Поиск'
      });


Плейсхолдер для отдельних разрешений:

      new CustomSearch({
         placeholder: {
            desktop: 'Поиск на ПК',
            tablet: 'Поиск на планшете',
            mobile: 'Поиск на телефоне',
         }
      });
